# 📊 Power BI Sales Dashboard

## 📌 Overview
This project is an interactive Power BI Sales Dashboard built using Power BI Desktop. It analyzes sales performance, customer information, product performance, and order trends.

## 🛠️ Tools Used
- Power BI Desktop
- Power Query
- DAX
- CSV Dataset

## 📂 Dataset
- Customers.csv
- Orders.csv
- Products.csv

## 📈 Dashboard Features
- Sales Overview
- Customer Analysis
- Product Analysis
- Order Insights
- Interactive Filters and Visualizations
